MAPFBenchmarks.download_benchmark_mapf — Method
download_benchmark_mapf(series, instance[; buckets])

Download a map and scenario from https://movingai.com/benchmarks/.

Arguments

  • series: the family of maps (eg. "street")
  • instance: the map id (eg. "Berlin_0_256")
  • buckets: the set of buckets to consider in the scenario (eg. 1:10)
